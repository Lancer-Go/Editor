# 🎨 HTML 原型可视化编辑工具

不写代码也能修改 HTML 原型！用浏览器打开，点选元素，直接编辑文字、调样式、拖位置，内置丰富的组件库支持一键拖拽插入。全程所见即所得，改完一键保存。

✨ **核心特性：**
- **文件与组件双面板**：内置超过 30 种常用 Web 组件，支持分类折叠与搜索过滤。
- **拖拽式插入**：直接从组件库拖拽组件到画布的指定位置，自动生成 DOM 结构。
- **智能对齐与尺寸调整**：拖拽元素时自动显示 5px 吸附辅助线；选中元素点击周边 8 个控制手柄调整宽高。
- **丰富的样式面板**：支持所见即所得修改排版、颜色、边距、圆角、背景。
- **原型多页交互**：选中组件后，可配置项目内 HTML 页面的快速跳转链接。

---

## 🚀 快速启动

### 方式一：双击启动（推荐）

双击 `start.bat`，浏览器会自动打开编辑器。

> 首次运行会自动安装依赖，需要等待几秒。

### 方式二：命令行启动

```bash
cd 编辑工具
npm install    # 仅首次需要
npm start      # 启动服务
```

启动后浏览器自动打开 `http://localhost:3456`。

## 📖 使用说明

### 1️⃣ 打开文件

在左侧面板顶部输入包含 HTML 文件的**目录路径**，点击「打开」。
内置了**目录记忆**功能，下次启动会自动展开你上次编辑的文件夹。

然后在「最近打开」或文件树中点击要编辑的 `.html` 文件，中间画布会加载预览。

### 2️⃣ 插入新组件

切换到左侧的「📦 组件」标签页，可以搜索或折叠寻找所需组件。
**将所需的组件拖拽**进入中间的画布区域，会在你鼠标释放的位置（就近元素之后）插入该组件。

### 3️⃣ 选择与调整大小

- **单击**画布中的任意元素 → 显示 8 个紫色调整手柄和蓝色边框
- **拖拽控制手柄** → 向四个方向或对角调整元素的宽度/高度
- 悬停元素时会有虚线提示

### 4️⃣ 编辑文字与链接

- **双击** 元素 → 直接在页面上打字修改文本内容
- **配置链接** → 选中可点击元素（如 `<a>` 按钮），在右侧面板的底部选择「🔗 页面跳转」，链接到项目内的其他 HTML 页面。

### 4️⃣ 调整样式

右侧属性面板可修改：

| 类别 | 可调项 |
|------|--------|
| 排版 | 字号、行高、字重、文字对齐 |
| 颜色 | 文字颜色、背景颜色 |
| 间距 | 外边距（margin）、内边距（padding） |
| 尺寸 | 宽度、高度 |
| 边框 | 圆角、边框样式 |
| 显示 | display 模式、透明度 |

### 6️⃣ 移动元素与智能对齐

选中元素后，**按住鼠标拖拽**即可移动位置。
拖动过程中如果靠近其他元素的边缘或中心线，会自动显示**红色辅助线**并在 5px 距离内发生**智能吸附对齐**。

### 7️⃣ 保存

点击右上角「保存」按钮或按 `Ctrl + S`。

> 保存时会自动创建 `.backup` 备份文件，放心操作。

### 8️⃣ 设备预览

底部栏切换不同屏幕尺寸：💻 桌面 / 📱 平板 / 📲 手机 / ↔ 自适应

## ⌨️ 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + S` | 保存 |
| `Ctrl + Z` | 撤销 |
| `Ctrl + Y` | 重做 |
| `Delete` | 删除选中元素 |
| `Esc` | 取消选中 |
| 双击元素 | 进入文字编辑 |

## ⚠️ 注意事项

- 需要 **Node.js** 环境（[下载地址](https://nodejs.org/)）
- 端口默认 `3456`，如被占用可修改 `server.js` 第 6 行
- 编辑器通过 iframe 加载原型，原型中引用的 CSS/JS/图片等资源会自动代理加载
