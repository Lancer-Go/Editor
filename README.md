# 🎨 HTML 原型可视化编辑工具

不写代码也能修改 HTML 原型！用浏览器打开，点选元素，直接编辑文字、调样式、拖位置，改完一键保存。

---

## 🚀 快速启动

### 方式一：双击启动（推荐）

双击 `start.bat`，浏览器会自动打开编辑器。

> 首次运行会自动安装依赖，需要等待几秒。

### 方式二：命令行启动

```bash
cd 编辑工具
npm install    # 仅首次需要
npm start      # 启动服务
```

启动后浏览器自动打开 `http://localhost:3456`。

---

## 📖 使用说明

### 1️⃣ 打开文件

在左侧面板顶部输入包含 HTML 文件的**目录路径**，点击「打开」。

```
示例路径：C:\我的项目\原型
```

然后在文件列表中点击要编辑的 `.html` 文件，即可在中间画布加载预览。

### 2️⃣ 选择元素

- **单击** 画布中的任意元素 → 蓝色边框高亮 → 右侧面板显示属性
- **悬停** 元素 → 虚线提示可选中

### 3️⃣ 编辑文字

- **双击** 元素 → 直接在页面上打字修改
- 或在右侧面板的「文字内容」区域输入

### 4️⃣ 调整样式

右侧属性面板可修改：

| 类别 | 可调项 |
|------|--------|
| 排版 | 字号、行高、字重、文字对齐 |
| 颜色 | 文字颜色、背景颜色 |
| 间距 | 外边距（margin）、内边距（padding） |
| 尺寸 | 宽度、高度 |
| 边框 | 圆角、边框样式 |
| 显示 | display 模式、透明度 |

### 5️⃣ 移动元素

选中元素后，**按住鼠标拖拽**即可移动位置。

### 6️⃣ 保存

点击右上角「保存」按钮或按 `Ctrl + S`。

> 保存时会自动创建 `.backup` 备份文件，放心操作。

### 7️⃣ 设备预览

底部栏切换不同屏幕尺寸：💻 桌面 / 📱 平板 / 📲 手机 / ↔ 自适应

## ⌨️ 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + S` | 保存 |
| `Ctrl + Z` | 撤销 |
| `Ctrl + Y` | 重做 |
| `Delete` | 删除选中元素 |
| `Esc` | 取消选中 |
| 双击元素 | 进入文字编辑 |

## 📁 项目结构

```
编辑工具/
├── server.js      # 后端服务（文件读写 API）
├── editor.html    # 前端编辑器界面
├── package.json   # 依赖配置
├── start.bat      # 双击启动脚本
└── README.md      # 本文档
```

## ⚠️ 注意事项

- 需要 **Node.js** 环境（[下载地址](https://nodejs.org/)）
- 端口默认 `3456`，如被占用可修改 `server.js` 第 6 行
- 编辑器通过 iframe 加载原型，原型中引用的 CSS/JS/图片等资源会自动代理加载
